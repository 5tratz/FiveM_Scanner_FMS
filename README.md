# FiveM Scanner (FMS)

<img src="https://i.ibb.co/Q772zc1W/Screenshot-2.jpg" alt="">

Advanced security tool for detecting potential backdoors and malicious code in FiveM server files.

<p align="center"> <img src="https://img.shields.io/badge/license-Proprietary-blue" alt="License"> <img src="https://img.shields.io/badge/platform-Windows-lightgrey" alt="Platform"> <img src="https://img.shields.io/badge/python-3.8%2B-green" alt="Python"> <img src="https://img.shields.io/badge/version-1.0.0-brightgreen" alt="Version"> </p>

<b>Features</b>:

- Multi-threaded scanning engine.
- Real-time statistics with three-panel layout.
- Export to multiple formats (TXT, JSON, HTML).
- Professional HTML Reports with severity visualization.
- Scan History tracking with 50 scan memory.
- Custom pattern support with regex validation.
- Import/Export patterns to share with others.
- Whitelist management system.
- Dark/Light themes with hover effects.
- UI Size scaling (Small/Medium/Large).
- Multi-language support (English/中文).
- Configurable text editor.
- Sortable columns with intelligent sorting.
- Auto-expanding match details with context.
- Reviewed/Unreviewed tracking system.
- Sensitivity control (10-100%).
- Deep scan mode with all file types.
- Quick scan mode for fast results.
- Thread-safe scanning operations.
- Comprehensive error logging.
- Settings stored in AppData.
- Auto-update checker.
- Report Issue directly to GitHub.
- Data directory quick access.
- Reset ALL settings to defaults.
